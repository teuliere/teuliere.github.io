Le circuit autour de St Martin

Le descriptif complet de ce chemin de randonnée est donnée dans ce PDF.
Un prospectus décrivant le chemin de randonnée existe également

Ce chemin de randonnée permet de voir

  • le pont de l'âne
  • le viaduc de Rebuzo
  • Le village de St Martin depuis plusieurs point de vue
  • le camp grant
  • le plateau de planèse et son point de vue (descendre dans la grotte bergerie)
  • le col de Saint Martin
  • la sources de Saint Martin
